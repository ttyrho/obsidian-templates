# Obsidian Templates
This repository contains the templates I use to build up my knowledge using Obsidian.

## Plugins I use
These are ther plugins I currently use and not the ones I always use.
- [Callout Manager](https://github.com/eth-p/obsidian-callout-manager)
- [Dataview](https://github.com/blacksmithgu/obsidian-dataview)
- [Excalidraw](https://github.com/zsviczian/obsidian-excalidraw-plugin)
- [Linter](https://github.com/platers/obsidian-linter)
- [Spaced Repetition](https://github.com/st3v3nmw/obsidian-spaced-repetition)
- [Templater](https://github.com/SilentVoid13/Templater)
